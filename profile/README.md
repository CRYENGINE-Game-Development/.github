## What is CRYENGINE?

CRYENGINE is a powerful, all-in-one 3D game engine solution proven in production by Crytek, ready to be licensed by third-party developers to bring their creative vision to life [citation:14]. Titles like Hunt: Showdown, the Crysis series, and The Climb were all developed with CRYENGINE [citation:1][citation:14]. Its core features include a production-proven Sandbox editor, a low-overhead renderer, and advanced AI and character animation systems [citation:2][citation:8].

Infrastructure teams building AAA games and applications benefit from CRYENGINE's fully-featured toolset and source code access. System administrators appreciate the "what you see is what you play" (WYSIWYP) system, which allows for real-time editing and iteration [citation:1].

## Installation Guide

### Prerequisites

- Windows 10/11
- Visual Studio 2015, 2017, or 2019 [citation:8]
- Git (optional, for source code)

### Step 1: Download the CRYENGINE Launcher

**Step 1:** Visit the CRYENGINE website.

**Step 2:** Download the CRYENGINE Launcher for Windows [citation:7].

**Step 3:** Run the launcher installer and follow the on-screen instructions.

### Step 2: Create an Account

**Step 1:** Launch the CRYENGINE Launcher.

**Step 2:** Create a new account or log in [citation:7].

### Step 3: Install an Engine Version

**Step 1:** In the launcher, go to **Library** > **My Engines**.

**Step 2:** Click **Add Engine** and select the desired version (5.5+ recommended) [citation:9].

**Step 3:** The launcher will download and install the engine.

**Note:** CRYENGINE 5.6 and beyond do not support development for 32-bit systems [citation:1].

### Step 4: Install from GitHub (Advanced)

**Step 1:** Visit the CRYENGINE GitHub repository.

**Step 2:** Download the desired release package.

**Step 3:** Extract and run `InstallEngine.bat` [citation:9].

### Step 5: Create a New Project

**Step 1:** In the launcher, go to the **Projects** view.

**Step 2:** Click **Add** and choose a project template.

**Step 3:** Select the installed engine version [citation:7].

**Step 4:** Enter the project name and location.

**Step 5:** Click **Create**.
